# walkingkooka-tree-expression-function-number
A collection of ExpressionFunction(s) that operate and return number results.


## Available functions

- abs(number)
- average(number...)
- ceil(number)
- count(object...)
- even(number)
- floor(number)
- isEven(number)
- isNumber(object)
- isOdd(number)
- max(number...)
- min(number...)
- number(object)
- odd(number)
- random()
- round(number)
- sign(number)
- sqrt(number)
- sum(number...)


Many more functions are outstanding and remain TODO.
