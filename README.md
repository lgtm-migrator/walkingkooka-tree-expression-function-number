# walkingkooka-tree-expression-function-number
A collection of ExpressionFunction(s) that operate and return number results.


## Available functions

- abs
- average
- ceil
- count
- floor
- max
- min
- number
- round
- sum


Many more functions are outstanding and remain TODO.
